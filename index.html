What it does

Built right, not just built.

⚡

Hardware interrupts

Buttons on D2 and D3 use INT0 and INT1 — they fire instantly regardless of what the main loop is doing.

▶️

Play / Pause toggle

One button to start or pause playback. Resumes exactly where it left off.

⏭️

Next track

Skip to the next song instantly. Loops back to track 1 after the last song.

🔁

Auto-advance

When a track finishes, the next one starts automatically. No action needed.

🛡️

Reliable debounce

Millis-based debounce with 300ms lockout. No double triggers, no phantom presses.

🔇

Non-blocking loop

Zero delay() calls in the main loop. Everything runs on millis() timers for smooth performance.

Storage

SD card setup.

  1. Format as FAT32

    Full format only — not quick format. Max 32GB card recommended.

  2. Create folder named 01

    DFPlayer looks for files inside numbered folders. The folder must be named exactly 01.

  3. Copy songs one by one via CMD

    DFPlayer plays by file system write order — not filename. Copy in sequence, never drag-and-drop all at once.

    format D: /FS:FAT32 /V:MUSIC
md D:\01
copy "song1.mp3" D:\01\001.mp3
copy "song2.mp3" D:\01\002.mp3
copy "song3.mp3" D:\01\003.mp3
copy "song4.mp3" D:\01\004.mp3

  4. Verify write order

    Run this command — files must appear as 001 → 002 → 003 → 004.

    dir D:\01 /OD

Help

Troubleshooting.

DFPlayer not found
Check TX→D10, RX→D11 via 1kΩ resistor, VCC=5V, SD card inserted.
Songs in wrong order
Reformat SD card and recopy files one by one in CMD. Never drag and drop.
First song skipped
File system write order is wrong. Redo the SD card setup from scratch.
Buttons feel laggy
Increase DEBOUNCE_MS from 300 to 350 in the code settings.
Audio crackling or noise
Add a 100µF capacitor between DFPlayer VCC and GND to filter power noise.
No sound at all
Check SPK_1 and SPK_2 connections. Speaker must be 4Ω or 8Ω — not a piezo buzzer.
Same song plays twice
Increase AUTOADVANCE_COOLDOWN from 3000 to 4000 in the code settings.
No sound on track 1 only
001.mp3 may be corrupt or copied in wrong order. Reformat SD and recopy.
